¿Qué es TestMania?

TestMania te permite crear y resolver tests sobre cualquier tema de forma muy sencilla. Puedes generar preguntas con tu IA favorita y tener tus pruebas organizadas en una tabla con progreso, tiempo y acciones rápidas.

Crear un test con IA

  1. En la sección Crea tu test, escribe la temática del test y elige cuántas preguntas quieres.
  2. Selecciona una plataforma de IA en el desplegable: ChatGPT, DeepSeek, Copilot, Gemini, Grok, Perplexity, Mistral o Claude.
  3. Pulsa Abrir IA. Se abrirá la plataforma elegida en una nueva pestaña y el prompt necesario se copiará automáticamente al portapapeles.
  4. Pega (Ctrl+V) ese texto en la IA y espera la respuesta.
  5. Copia todo el código JSON que te devuelva la IA y pégalo en el cuadro «Código del test (JSON)».
  6. Pulsa Crear Test. Si ya existe un test con el mismo título, se renombrará automáticamente (por ejemplo, «Mi test (2)»).
Consejo: la temática puede ser general o muy específica (ej.: capitales de África, discos de los Beatles, reglamento de fútbol…).

Formato del JSON

El JSON debe tener este esquema básico:

{
  "titulo": "Cultura general",
  "preguntas": [
    { "texto": "¿Capital de Francia?", "opciones": ["Madrid","Roma","Lisboa","París"], "correcta": 4 },
    { "texto": "¿Autor de Don Quijote?", "opciones": ["Cervantes","Lope","Quevedo","Góngora"], "correcta": 1 }
  ]
}
  • correcta indica la posición (1, 2, 3 o 4) de la respuesta válida.
  • Debe haber exactamente 4 opciones por pregunta.
  • El título puede ajustarse automáticamente si hay duplicados.

Tabla de tests y acciones

En la tabla principal verás cada test con sus aciertos, progreso, tiempo invertido y un desplegable de acciones por fila. Las acciones disponibles son:

  • Exportar Test a PDF — genera un PDF listo para imprimir.
  • Descargar JSON del Test — guarda el test en un archivo .json.
  • Copiar JSON del Test — copia el JSON al portapapeles.
  • Reiniciar Test — borra tus respuestas y el tiempo de ese test (solo localmente).
  • Eliminar Test — solo aparece para tests personalizados (los de ejemplo no se pueden borrar).

Exportar a PDF

La exportación a PDF crea un documento con:

  • Título del test.
  • Cada pregunta y sus 4 opciones, con salto de línea automático para textos largos.
  • Marca de agua TestMania.es en cada página.
  • Una sección final de respuestas correctas (número, letra y texto de la opción).

El archivo se descarga con el nombre del test y extensión .pdf.

¿Dónde se guardan mis tests?

Tus tests personalizados se guardan únicamente en tu navegador usando almacenamiento local (localStorage). Si borras los datos del navegador, usas otro navegador o cambias de dispositivo, no aparecerán automáticamente. Exporta a JSON para hacer copias de seguridad y poder importarlos más tarde.

Preguntas frecuentes

¿Puedo crear un test sin usar JSON?

De momento la creación es mediante JSON generado (o escrito) por ti. Estamos valorando añadir asistentes más visuales.

¿Qué pasa si ya existe un test con el mismo título?

La app lo renombrará automáticamente (p. ej., «Mi test (2)») para evitar duplicados.

¿Cómo importo un test que exporté antes?

Abre la sección Crea tu test, pega el contenido del archivo .json en el cuadro de texto y pulsa Crear Test.

¿Se comparten mis tests con otras personas?

No. Tus tests viven en tu navegador, a menos que exportes y compartas el JSON.

Privacidad

TestMania no sube tus tests a ningún servidor. Toda la información se mantiene localmente salvo que decidas exportarla o compartirla manualmente.

Problemas comunes

«Formato de JSON no válido» al crear un test

Revisa que el JSON esté completo y bien formado: comas, llaves y corchetes en su sitio; 4 opciones por pregunta; y correcta entre 1 y 4.
