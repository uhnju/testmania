¿Qué es TestMania?

TestMania te permite crear y resolver tests sobre cualquier tema de forma sencilla.

¿Dónde se guardan mis tests?

Tus tests se guardan únicamente en tu navegador usando almacenamiento local (localStorage). Si borras los datos del navegador o cambias de dispositivo, tus tests personalizados no estarán disponibles automáticamente.

Exportar e importar tests

Exportar e importar tests

Puedes exportar tus tests en formato JSON con el icono 💾 en la tabla.

Luego podrás importarlos de nuevo copiando ese JSON en el formulario de creación.

Exportar un test

  1. Abre la página principal y localiza tu test en la tabla.
  2. Pulsa el botón 💾 (Descargar JSON) en la fila del test.
  3. Se descargará un archivo .json con todas las preguntas y respuestas.

Importar (crear desde un JSON)

  1. Ve a la sección Crea un nuevo test con IA (o creación manual).
  2. Pega el contenido del JSON en el cuadro “Código del test (JSON)”.
  3. Pulsa Crear Test para añadirlo a tu lista.
Formato esperado del JSON (ver ejemplo) 
{
  "titulo": "Cultura general",
  "preguntas": [
    { "texto": "¿Capital de Francia?", "opciones": ["Madrid","Roma","Lisboa","París"], "correcta": 4 },
    { "texto": "¿Autor de Don Quijote?", "opciones": ["Cervantes","Lope","Quevedo","Góngora"], "correcta": 1 }
  ]
}

Preguntas frecuentes

¿Puedo crear un test sin usar JSON?

De momento la creación es mediante JSON. Estamos valorando otras opciones.

¿Qué pasa si ya existe un test con el mismo título?

La aplicación puede renombrarlo automáticamente (p. ej., “Mi test (2)”) para evitar duplicados.

¿Se comparten mis tests con otras personas?

No. Los tests son privados y viven en tu navegador, a menos que exportes y compartas el JSON.

Privacidad

TestMania no sube tus tests a ningún servidor. Toda la información se mantiene localmente salvo que decidas exportarla o compartirla manualmente.
